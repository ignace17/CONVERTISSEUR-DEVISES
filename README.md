# 💰 Convertisseur de Devises 

Un espace web dynamique qui permet de convertir des monnaies internationales en temps réel. Ce projet met en pratique la récupération de données financières via une API externe.

### 🚀 Fonctionnalités
- **Taux en temps réel** : Connexion à l'API ExchangeRate pour des données à jour.
- **Conversion Instantanée** : Calcul rapide entre plusieurs devises (EUR, USD, XOF, etc.).
- **Interface Intuitive** : Design épuré et responsive pour une utilisation sur mobile et PC.

### 🛠️ Technologies utilisées
- **HTML5 / CSS3** : Structure et design moderne.
- **JavaScript (ES6)** : Logique de calcul et gestion du `fetch`.
- **API REST** : Consommation de données JSON via ExchangeRate-API.

### ⚙️ Installation
1. Cloner le dépôt.
2. Ouvrir `index.html` avec Live Server.
3. Ajouter sa propre clé API dans `script.js`.
